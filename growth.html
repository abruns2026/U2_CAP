GROWTH

"Jumping George"

This app (U3 Project) represents my growth in coding interactive games and using movement controls, collision detection, and scoring systems. In my game, the user controls a monkey using the arrow keys to jump across leaf platforms, collect bananas, and dodge tigers. The user wins once they collect 15 bananas! The hardest part about this project was definitely making the character movement feel smooth, as well as making sure the monkey could correctly interact with the platforms, bananas, and tigers. I got past these obstacles by testing different variables and functions, adjusting my code multiple times, and asking Ms. Luna for help when I got stuck! If I were to improve anything about this project, I would add different levels/achievements, different obstacles, and power-ups to make the game even more interactive and exciting for users.

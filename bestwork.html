BEST WORK

"SHOWBOXD App Design"

I would consider this project my best work, as it combines creativity and computer science in a personal way. I created a TV show version of Letterboxd where users can track, rate, and review shows they watch. It stands out because it solves a real problem and feels like a real app people would definitely use. I’m proud of this project because I turned an idea into a functional, visually appealing result. It was quite challenging because I had to balance coding with design and make sure everything worked smoothly. The worst part was making sure every button used had another page linked to it. I remember there being so many onEvents in the code, and it was hard to keep track of what was and wasn’t missing. With this project, I used skills such as creativity, coding, and lots of patience to create a project that reflects both my interests and my growth in computer science.
